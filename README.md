# GoProCam
Live video and audio transmission project with a GoPro from a remote Raspberry Pi.

## Dependencies
- [FFMPEG](https://ffmpeg.org/)
- [GoPro-Py-API](https://github.com/KonradIT/gopro-py-api)
- [GoProWifiHack](https://github.com/KonradIT/goprowifihack)
- [SRT Protocol](https://github.com/Haivision/srt/)
- [Nginx](https://www.nginx.com/)
